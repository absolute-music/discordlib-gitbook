# login

## How to get started?

### login

| Parameter | Description | Required |
| :--- | :--- | :--- |
| discord bot token | to start the bot | yes |
| game | set bot's game | no |

### Example

```
let dl = require("discord.lib")
dl.login("Token", "game")
```
