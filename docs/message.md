# message

### Commands

| Parameter | Description | Required |
| :--- | :--- | :--- |
| command | Set the command\(includes prefix\) | yes |
| message | what will the bot reply to you | yes |

## Example

```
const dl = require("discord.lib");
dl.sendMessage("help", "reply")
dl.login("Token", "game")
```
