# serverGreeting

You could greet the server owner when your bot was invited to a server using this

### Param

| Parameter | Discription | Required |
| :--- | :--- | :--- |
| message | message to greet | yes |

### Example

```javascript
const dl = require("discord.lib")
dl.serverGreeting("Hi there!")
```
