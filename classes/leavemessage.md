# leaveMessage

### Parameters

| Parameter | Description | required |
| :--- | :--- | :--- |
| channel | channel id | yes |
| message | welcome message | yes |

### Example

```text
const dl = require("discord.lib");

dl.leaveMessage("633412217844269045", "Welcome!")
dl.login("token", "I am the best bot!")
```
